# Project ORIGIN Research Rules

The purpose of this repository is to investigate recurring symbolic structures using evidence.

## Every finding must be labeled as one of the following:

### Observation

Directly supported by archaeological evidence.

### Hypothesis

A possible explanation supported by some evidence.

### Speculation

An idea that is interesting but currently lacks sufficient evidence.

---

## Research Principles

1. Evidence before conclusions.

2. Record sources whenever possible.

3. Never alter historical data.

4. Geometry is analyzed before symbolism.

5. Similarity does not automatically imply common origin.

6. Remain open to new evidence.

7. Any conclusion should be capable of being challenged.

---

Project ORIGIN seeks understanding through careful comparison, transparency, and repeatable methods.
